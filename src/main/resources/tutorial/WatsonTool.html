Tweets Refactor


Se você já possui arquivos de coleta que foram gerados através de outros softwares, porém esses arquivos possuem dados como: Retweets, datas, geolocalização, nome de usuário, etc. Você pode executar uma refatoração nesses arquivos utilizando o "Refactor" que se encontra no menu "Ferramentas". você pode processar um único arquivo ou até mesmo vários arquivos de uma vez só! Para processar somente um arquivo basta manter a opção "Lote de Arquivos" desmarcada e o contrário para processar uma pasta com vários arquivos. O utilitário aceita arquivos no formato ".csv" e ".txt" e deve usar o "|" (pipeline) como delimitador. Você também tem a opções como: remover os links, remover acentuação , remover retweets e extrair tweet puro dos seus arquivos, basta marcar as respectivas opções.


Executando o refactor:
  1. Selecione um arquivo ou diretório de entrada, conforme seu objetivo.
  2. Selecione o diretório onde deseja salvar o novo arquivo. (As saídas serão geradas no formato ".txt").
  3. Selecione uma ou mais opções de refatoração.
  4. Clique no botão "Processar"
