Coleta Retroativa


A opção de coleta retroativa já vêm marcada por padrão, infelizmente a API REST fornece apenas 1% de acesso ao banco de dados do Twitter, sendo possível recuperar tweets com no máximo 7 dias (Mais recentes). Você pode selecionar dentro deste limite, quantos dias deseja fazer a coleta retroativa. Informe o termo de busca no campo "Palavra-Chave" e clique em "Coletar". Espere a coleta finalizar, logo após, uma mensagem é exibida, onde você pode abrir o diretório onde os arquivos foram armazendados.

Você também tem a opção de remover endereços web (link) dos tweets coletados, basta acessar o menu "Configurações" e marcar a opção "Remover Links".
